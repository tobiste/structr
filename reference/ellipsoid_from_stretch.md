# Ellipsoid tensor from principal stretches

Return diagonal tensor defined by magnitudes of principal stretches

## Usage

``` r
ellipsoid_from_stretch(x = 1, y = 1, z = 1)
```

## Arguments

- x, y, z:

  numeric. Magnitudes of principal stretches

## Value

object of class `"ellipsoid"`

## Details

The eigenvalues \\\lambda\\ of the deformation matrix are the quadratic
forms of the principal stretches \\s\\ (\\s = 1 + \epsilon = l/l_0\\).

## See also

Other ellipsoid:
[`ellipsoid-class`](https://tobiste.github.io/structr/reference/ellipsoid-class.md),
[`ellipsoid-params`](https://tobiste.github.io/structr/reference/ellipsoid-params.md)

## Examples

``` r
el <- ellipsoid_from_stretch(4, 3, 1)
principal_stretch(el)
#> S1 S2 S3 
#>  4  3  1 

lapply(seq.int(nrow(hossack1968)), function(i) {
  ellipsoid_from_stretch(hossack1968[i, 3], hossack1968[i, 2], hossack1968[i, 1])
})
```
