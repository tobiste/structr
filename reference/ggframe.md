# Stereoplot Perimeter

Adds a frame to the stereographic projection

## Usage

``` r
ggframe(n = 10000, color = "black", fill = NA, lwd = 1, ...)
```

## Arguments

- n:

  integer. Resolution of line.

- color, fill, lwd:

  Graphical parameters

- ...:

  optional graphical parameters passed to
  [`ggplot2::geom_polygon()`](https://ggplot2.tidyverse.org/reference/geom_polygon.html)
