# Sorting StraboSpot Projects

Sorts `strabo` objects in the order determined by one or more other
objects, typically columns in the `data` data.frame. This will also sort
the `planar` and `linear` objects.

## Usage

``` r
# S3 method for class 'strabo'
sort_by(x, y, ...)
```

## Arguments

- x:

  object of class `"strabo"`

- y:

  Variables to sort by. Typically a column in `x$data` data.frame. To
  sort by more than one column, take the form `list(g1, g2, ...)`.

- ...:

  Additional arguments, typically passed on to
  [`order()`](https://rdrr.io/r/base/order.html).

## Value

A sorted version of `x`.
