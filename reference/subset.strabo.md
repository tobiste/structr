# Subsetting StraboSpot Projects

Retursn a subset from `"strabo"` objects which meet conditions.

## Usage

``` r
# S3 method for class 'strabo'
subset(x, ..., ds = c("data", "spots", "tags"))
```

## Arguments

- x:

  object of class `"strabo"`

- ...:

  arguments to be passed to
  [`subset()`](https://rdrr.io/r/base/subset.html)

## Value

An object of class `"strabo"` containing just the selected rows and
columns of `"x"`.

## See also

[`subset()`](https://rdrr.io/r/base/subset.html)

Other strabo:
[`strabo`](https://tobiste.github.io/structr/reference/strabo.md)
