# Colorblind Color Palette (Discrete) and Scales

Colorblind Color Palette (Discrete) and Scales

## Usage

``` r
colorblind_pal(n)
```

## Arguments

- n:

  number of colors to extract. Gives a warning when `n` exceeds 8.

## Value

character vector with color values

## Examples

``` r
colorblind_pal(3)
#> [1] "#000000" "#E69F00" "#56B4E9"
```
