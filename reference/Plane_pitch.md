# Pitch of a line of a plane

Pitch of a line of a plane

## Usage

``` r
Plane_pitch(p, l)
```

## Arguments

- p:

  object of class `"Plane"`

- l:

  object of class `"Line"`

## Value

Pitch angle in degrees

## Examples

``` r
Plane_pitch(Plane(90, 70), Line(175, 13)) # 13.9
#> [1] 13.91292
```
