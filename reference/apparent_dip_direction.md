# Apparent dip direction

Apparent dip direction

## Usage

``` r
apparent_dip_direction(p, apparent_dip)
```

## Arguments

- p:

  object of class `"Plane"`

- apparent_dip:

  angle in degrees

## Value

Azimuth in degrees

## Examples

``` r
apparent_dip_direction(Plane(63, 45), 10)
#> [1] 145.8929
```
